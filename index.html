Super Duper Chores

Check out the GitHub repo here!

Super Duper Chores was created as a backend group project. We used Node.js, Express, PostgreSQL, HTML, and CSS to build this web app.

Users are able to login and create teams (for work, home, etc.) consisting of members who are also registered users. Teams can then create, track, update, and comment on various team-specific chores and tasks.

The inspiration behind this app stems from my own personal struggles with keeping tasks around the house organized!

The Dad Joke Bracket Challenge is a React Frontend solo project that allows users to pick their favorite dad joke. This app was built using React, CSS, and JSX.

The app fetches information from a Dad Joke API and randomly generates matchups in a clickable tournament style bracket. Picture the NCAA bracket but for Dad Jokes!

The inspiration for this web app was to give my dad a chuckle.



Sistema de Alerta Temprana

Check out the GitHub repo here!

Sistema de Alerta Temprana is a full-stack capstone project that monitors and tracks water levels at various locations along Guatemalan rivers. This site will eventually be used by Galileo University in Guatemala City to gauge and predict locations at risk of flooding throughout the country. We built this site with HTML, CSS, React, React-Bootstrap, Node.js, Express, PostgreSQL, and Chart.js.

Users are able to view up to date water levels, images, and locations for each sensor site on rivers throughout Guatemala. New information for each river monitor station is updated every 15 minutes.
